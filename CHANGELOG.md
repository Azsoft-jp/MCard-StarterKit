# Changelog

## 0.1.0

Initial public clean-room starter kit.

### Included

- Local Node.js dashboard.
- Profile editor and sample profiles.
- Media and animation preparation tools.
- Browser-native media import path.
- Profile-driven frame builders.
- Web Bluetooth opt-in transport.
- Local notification parser and retry tools.
- Emulator notification simulator.
- Windows BLE peripheral source sample.
- Local synthetic OTA package verifier.
- Transfer-time estimator.
- Workspace export/import and migration.
- Hardware planning templates.
- Public documentation and clean-room audit.
