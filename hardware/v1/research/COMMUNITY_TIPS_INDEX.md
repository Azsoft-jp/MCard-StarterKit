# Community tips index

Community references are optional implementation leads, never manufacturing or
safety authority. Official Help, Documentation, Support, datasheets, standards,
and application notes win on conflict.

| Topic | Community lead | Use |
|---|---|---|
| JLC/KiCad export | [KiCad JLCPCB Tools](https://github.com/Bouni/kicad-jlcpcb-tools) | Evaluate BOM/CPL convenience; verify output against JLC Help |
| LCSC library import | [easyeda2kicad.py](https://github.com/uPesy/easyeda2kicad.py) | Candidate symbol/footprint import; manually verify all geometry |
| PCB as code | [atopile docs](https://docs.atopile.io/) | Future declarative circuit option, not adopted for V1 scaffold |
| PCB as code | [tscircuit](https://github.com/tscircuit/tscircuit) | Future rapid prototype option; mature KiCad review still required |

No forum/blog numerical rule should be copied into constraints without an
official source. Add author, URL, access date, claimed tip, test result, and
official cross-check before accepting any future community reference.
